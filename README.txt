All of the models are created in jupyter notebooks. There is a folder for each dataset and a unique file in each folder for each model. To use the models, just download the folders,
double check to make sure the bank-data.csv and credit_card_default.csv are in their proper folders, and open the .ipynb files in jupyter notebook. After this, you can run different
pieces of code or all of the code together.
